## [2026-06-03 09:06] python_syntax — 학습 공백 진단 (어제 끝점 ↔ 데코레이터)

세션 시작 시 어제 학습 로그 보고 레벨 3 코딩(`@log_calls` 데코레이터) 출제했으나 갭이 큼. 분해 결과 정리.

**어제까지 정착된 것**:
- `*args`, `**kwargs` **모으기** (정의 시) — merge_dicts, make_record 로 실습
- `*args`, `**kwargs` **풀기** (호출 시) — `greet(**info)` 로 학습
- 매개변수 vs 인자 용어

**어제 메모 끝에 예고만 됨 (미실습)**:
- ⚠️ **relay pattern**: 모았다가 그대로 풀어서 다른 함수에 넘기기 — `func(*args, **kwargs)`

**데코레이터 문제가 추가로 요구한 미학습 토픽들**:
1. 함수가 **일급 객체** (함수를 인자로 받고, 함수를 반환)
2. **클로저** (내부 함수 `wrapper` 가 외부 변수 `func` 기억)
3. `@데코레이터` 문법 (= `add = log_calls(add)` 의 syntactic sugar)
4. `func.__name__` 같은 함수 객체 속성

→ 이 4개는 "`*args, **kwargs` 의 다음 단계" 가 아니라 **별도의 큰 토픽 (함수를 다루는 함수)**.

**다음 스텝**:
1. relay pattern 만 단독으로 레벨 3 실습 (어제 메모 마지막 줄에 충실)
2. 그 후 데코레이터를 새 토픽으로 분리해서 진입 (일급 객체 → 클로저 → `@` 문법 순서)

## [2026-06-03 09:50] python_syntax — 부족한 것 진단 (4단 무너진 후)

세션 내내 사다리 1~4단 진행. 4단에서 부하 폭주로 무너짐 ("울 것 같다", "구해줘"). 정착된 것과 빈 곳을 명확히 분리.

### ✅ 이미 정착된 것 (계속 보유)
- 함수가 일급 객체 (괄호 X = 객체 자체, 괄호 O = 호출 결과)
- 클로저 = 도시락통 (`make_multiplier` 12·28·30 정확히 풀어냄)
- 데코레이터 = 함수를 감싸는 함수 (값 추적 정확)
- `*args/**kwargs` 모으기/풀기 (어제 학습)
- 객체/이름표 구분 (객체 = 메모리의 실체, 변수 = 이름표)
- 매개변수 vs 인자

### ❌ 부족한 것 (다음 세션 우선)

1. **f-string 포매팅** — 한 번도 명시적으로 학습 안 함
   - `f"...{변수}..."` 의 치환 메커니즘
   - 특히 `{func.__name__}` 같이 **속성 접근**이 들어간 형태
   - 다중 치환 동시 시뮬레이션 (한 줄에 `{...}` 3개)
   - **이게 4단 무너짐의 직접 원인**

2. **함수의 속성** (`func.__name__`)
   - "함수 객체가 라벨을 들고 다닌다" 개념
   - `.`(dot)으로 접근 가능 = 함수도 속성을 가진 진짜 객체

3. **출력 형식 추적 스킬 (별도 스킬)**
   - "이 코드 실행 시 화면에 정확히 무엇이 찍히는가" 의 mental simulation
   - 클로저/데코레이터 이해와 **별개의 근육**

4. **데코레이터 후 이름표 정체 변화** (⚠️ 약점)
   - `@log_calls` 가 붙으면 `add` 가 더 이상 원래 `add` 가 아니라 `wrapper`
   - 두 번 짚어줬는데도 또 막힘 → **반복 출제 필요**

5. **return ≠ print** (⚠️ 약점, 세 번째 같은 실수)
   - "함수가 값을 반환" ≠ "그 값이 화면에 출력"
   - 누가 받아서 `print` 로 감싸지 않으면 안 보임
   - 직전 문제에서도 짚었는데 다시 같은 실수 → **약점 토픽 등록**

### 🎯 다음 세션 진입 전략
1. f-string 단독 미니 학습 (`f"안녕 {name}"` → `f"{객체.속성}"` → 다중 치환)
2. 함수 속성 (`__name__`) 짧게 도입
3. 출력 형식 예측 연습 5문제 (클로저 다 빼고 print 만)
4. 그 다음에 데코레이터 4단 문제 다시 시도
5. return vs print 는 매 세션 1문제씩 끼워넣어 정착시킬 것

### 💬 정서 메모
